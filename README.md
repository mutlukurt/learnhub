## ğŸš€ Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation
```bash
# Clone the repository
git clone https://github.com/mutlukurt/learnhub.git

# Navigate to project directory
cd learnhub

# Install dependencies
npm install

# Start development server
npm start
```

```bash

npm run build

# Deploy to GitHub Pages

```
